#### GitHub Actions 工作流

```yaml
# .github/workflows/ci.yml
name: CI Pipeline

on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  # 1. 静态代码分析
  static-analysis:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: SonarQube Scan
        run: |
          # 后端 Go 代码分析
          golangci-lint run ./...
          # iOS 代码分析 (需要 macOS)
          - name: SwiftLint
            run: swiftlint

  # 2. 单元测试
  unit-tests:
    runs-on: ubuntu-latest
    services:
      postgres:
        image: postgres:15
        env:
          POSTGRES_PASSWORD: test
      redis:
        image: redis:7
    steps:
      - uses: actions/checkout@v4
      - name: Go Backend Tests
        run: |
          go test -race -coverprofile=coverage.out ./...
          go tool cover -func=coverage.out
      - name: Upload Coverage
        uses: codecov/codecov-action@v3

  # 3. API 集成测试
  api-integration:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Run API Tests
        run: |
          newman run Postcards-API.postman_collection.json \
            --environment production.json

  # 4. 构建 Docker 镜像
  build:
    runs-on: ubuntu-latest
    needs: [unit-tests]
    steps:
      - uses: actions/checkout@v4
      - name: Build Docker Image
        run: |
          docker build -t postcards:${{ github.sha }} .
          docker push postcards:${{ github.sha }}

  # 5. 部署到 Staging
```

## 五、安全测试要点

### 5.1 认证授权测试

| ID | 测试项 | 方法 | 预期 |
|:---|:---|:---|:---|
| AUTH-01 | JWT 伪造 | 修改 Token payload | 签名验证失败，拒绝访问 |
| AUTH-02 | Token 盗用 | 使用他人 Token | 返回 403，禁止访问 |
| AUTH-03 | 权限提升 | 普通用户访问管理员接口 | 返回 403 |
| AUTH-04 | 会话固定 | 登录前后 Session ID | 必须变更 |
| AUTH-05 | 暴力破解 | 连续失败登录 | 触发账号锁定 |
| AUTH-06 | CSRF | 跨站请求 | Token 验证阻止 |

### 5.2 数据加密验证

| ID | 测试项 | 方法 | 预期 |
|:---|:---|:---|:---|
| CRYPTO-01 | 传输加密 | 抓包检查 HTTP | 全站 HTTPS |
| CRYPTO-02 | Token 存储 | 检查数据库 | 无明文密码 |
| CRYPTO-03 | 敏感字段 | 检查日志 | 无敏感信息明文 |
| CRYPTO-04 | OSS 访问 | 未授权访问 URL | 签名 URL 或拒绝 |
| CRYPTO-05 | 密码强度 | 弱密码注册 | 拒绝注册，提示加强 |
