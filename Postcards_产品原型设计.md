# Postcards - 产品原型设计文档

## 一、产品概述

**产品名称**: Postcards  
**定位**: 让旅行者"轻松记录、自动整理、美好回忆"的智能旅行日志工具  
**Slogan**: Every trip sends you a postcard.

---

## 二、信息架构

```
Postcards
├── 首页 (Home)
│   ├── 当前旅行卡片
│   ├── 地图足迹缩略图
│   └── 快速记录入口
├── 时间轴 (Timeline)
│   ├── 按旅行筛选
│   ├── 按日期浏览
│   └── 搜索与标签
├── 地图 (Map)
│   ├── 全球足迹热力图
│   ├── 单条旅行轨迹
│   └── 地点详情弹窗
├── 收藏夹 (Collection)
│   ├── 年度收藏册
│   ├── 精选明信片
│   └── 实体印刷订单
├── 我的 (Profile)
│   ├── 旅行统计
│   ├── 设置
│   └── 订阅管理
└── 记录流程 (Record Flow)
    ├── 拍照/导入
    ├── 自动识别地点+时间
    ├── 可选语音补充
    └── 生成明信片预览
```

## 四、关键用户流程

### 流程 1: 首次使用 → 完成第一条记录
```
下载 APP → 注册/登录 → 授予相册权限 → 
首页引导"开始你的第一次旅行" → 点击 FAB → 
选择拍照 → 拍摄 → 自动识别地点 → 
预览明信片 → 保存 → 首页显示第一条记录
```

### 流程 2: 日常记录（已有进行中的旅行）
```
打开 APP → 首页显示当前旅行 → 点击"继续记录"或 FAB → 
快速拍照 → 自动归类到今天 → 完成（3秒内）
```

### 流程 3: 回顾与分享
```
进入时间轴 → 浏览历史记录 → 点击单张明信片 → 
左右滑动浏览 → 点击分享 → 选择"寄出副本" → 
生成分享卡片 → 分享到微信/Instagram
```

### 流程 4: 生成年度收藏册
```
进入收藏夹 → 点击年度收藏册 → AI 自动精选最佳照片 → 
用户可替换/调整 → 确认排版 → 选择导出方式 → 
[数字版免费] / [实体印刷付费]
```

## 五、功能优先级 (MVP → V1 → V2)

| 优先级 | 功能模块 | 说明 |
|:---|:---|:---|
| **P0 (MVP)** | 拍照记录、自动时间地点、时间轴浏览、基础地图 | 核心闭环 |
| **P1 (V1)** | 语音转文字、标签系统、分享卡片、年度收藏册 | 体验提升 |
| **P2 (V1.5)** | AI 故事生成、多人协作、3D 地图轨迹、实体印刷 | 差异化功能 |
| **P3 (V2)** | 社交发现、旅行灵感推荐、积分成就系统 | 社区化 |

## 六、异常与边界情况

| 场景 | 处理方案 |
|:---|:---|
| 照片无 GPS 信息 | 提示用户手动选择地点，或 AI 图像识别地标 |
| 离线状态 | 本地存储，联网后自动同步，显示"待同步"标识 |
| 同一天多地点 | 时间轴内按地点自动分组，显示子标题 |
| 照片重复导入 | 哈希去重，提示"该照片已存在" |
| 旅行结束判定 | 连续 3 天无新记录，推送"结束这次旅行？" |
